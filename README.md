# ChatSphere 🤖

ChatSphere is an AI-powered chatbot application built using React, Node.js, Express.js, MongoDB, and Google Gemini API. It provides real-time AI conversations through a modern and responsive user interface.

## 🚀 Live Demo

https://chat-sphere-tc9q.vercel.app/

## 📂 GitHub Repository

https://github.com/Kapur2026/ChatSphere

## ✨ Features

* AI-powered chatbot using Google Gemini API
* Real-time conversations
* Modern and responsive UI
* Full-stack architecture
* MongoDB database integration
* Deployed with Vercel and Render

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### AI Integration

* Google Gemini API

## 📦 Installation

### Clone the repository

```bash
git clone https://github.com/Kapur2026/ChatSphere.git
cd ChatSphere
```

### Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file and add:

```env
MONGODB_URI=your_mongodb_uri
GEMINI_API_KEY=your_gemini_api_key
PORT=4002
```

Start the backend:

```bash
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

* Frontend: Vercel
